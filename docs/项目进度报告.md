## 3. 当前项目结构

```
泡泡堂/
├── server.js              # HTTP + WebSocket 入口
├── package.json           # 依赖: ws
├── src/backend/
│   ├── roomManager.js     # 房间 CRUD + 状态管理
│   ├── gameLoop.js        # 20tick/s 确定性游戏循环
│   ├── mapGenerator.js    # 4 种地图 + BFS 连通性验证
│   └── botAI.js           # 4 级难度 Bot
├── public/
│   ├── index.html         # 前端页面 + UI
│   ├── js/
│   │   ├── lib/phaser.min.js  # Phaser 3.60 本地化
│   │   ├── game.js            # 前端主程序（1400+ 行）
│   │   └── auto-start.js      # Agent 自动检测开战
│   └── assets/
│       ├── characters/    # 4 角色 × 3 动作精灵图
│       ├── items/         # 5 种道具图标
│       └── bgm.mp3        # 背景音乐
└── docs/                  # 需求/方案/总结文档
```

## 4. 技术栈

| 层级 | 技术 |
|------|------|
| 前端引擎 | Phaser.js v3.60（程序化纹理 + AI 精灵图） |
| 后端 | Node.js + `ws` 库 |
| 实时通信 | WebSocket（20tick/s 状态同步，100ms 广播） |
| 部署 | 局域网 `0.0.0.0:3000` + 内网穿透可用 |

## 5. 已实现功能清单

| 类别 | 功能 |
|------|------|
| 地图 | 13×11 网格，4 种主题，随机石头 + BFS 连通验证 |

| 观战 | 浏览器实时渲染 + 战斗日志侧边栏 |
