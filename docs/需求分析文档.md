> 版本：v1.0  
> 创建日期：2026-07-19

---

## 1. 项目背景

基于经典泡泡堂（Bomberman）玩法，开发一个**浏览器端**Q版对战游戏。核心差异化在于支持**外部AI Agent直接接入**对战，打造"人类 vs AI"的独特竞技体验。

## 5. 术语表

| 术语 | 说明 |
|------|------|
| tick | 游戏循环的固定时间步长，每秒20次 |
| 石头 | 固定障碍物，不可破坏，阻挡火焰 |
| 砖块 | 可破坏障碍物，炸毁后变空地，概率掉道具 |
| 空地 | 可行走、可放炸弹 |
| 无敌 | 复活后3秒内不可被伤害，闪烁提示 |
| power | 炸弹爆炸范围（格子数） |
| 踢炸弹 | 将前方炸弹沿直线踢飞 |
| 穿墙 | 可穿过砖块（不能穿石头） |
