# AI Q版泡泡堂 · 快速启动说明

> 从零到游戏可玩的完整步骤，预计耗时 **5分钟**。

---

## 方式一：快速启动（推荐）

### 第1步：进入项目目录

```bash
cd "E:\Desktop\光年船说\其他\泡泡堂"
```

### 第2步：安装依赖

```bash
npm install
```

> 仅需安装 `ws`（WebSocket库），约 1MB，速度很快。

### 第3步：启动服务器

```bash
npm start
```

启动后你会看到类似这样的输出（你的IP可能不同）：

```
🎮 AI Q版泡泡堂服务器已启动！
   端口: 7100
   监听: 0.0.0.0 (所有网络接口)

   👤 本机访问: http://localhost:7100
   🌐 局域网访问: http://26.228.246.39:7100
   📡 WebSocket: ws://26.228.246.39:7100/ws

   💡 其他AI Agent连接地址: ws://26.228.246.39:7100/ws
   💡 分享链接: http://26.228.246.39:7100
```

### 第4步：打开浏览器玩游戏

打开浏览器，访问：
```
http://localhost:7100
```

### 第5步：创建房间并邀请好友

1. 输入你的名字
2. 选择游戏模式（推荐"自由混战"）
3. 点击 **创建房间**
4. 记下4位房间号（如 `BV8C`）
5. 点击房间号可复制
6. 告诉好友：打开浏览器访问你的地址，输入房间号加入

### 第6步：添加Bot（可选）

在房间大厅点击 **添加** Bot，选择难度即可。

### 第7步：开始游戏

所有玩家点击 **准备** 后，房主点击 **开始游戏** 即可开战！

## 方式二：两个浏览器窗口本地测试（1人模拟多人）

如果你暂时没有朋友在线，可以用两个浏览器窗口自己测试：

### 步骤：

1. 按 **方式一** 启动服务器
2. 打开浏览器窗口A，访问 `http://localhost:7100`
3. 打开浏览器窗口B（按 `Ctrl+N` 新窗口），访问 `http://localhost:7100`
   - 输入名字 "玩家A"
   - 创建房间，记下房间号
3. 打开浏览器窗口B（按 `Ctrl+N` 新窗口），访问 `http://localhost:7100`
   - 输入名字 "玩家B"
   - 点击 **加入房间**
   - 输入窗口A的房间号
4. 两个窗口都点击 **准备**
5. 窗口A点击 **开始游戏**
6. 在窗口A按 `WASD` 移动、`空格` 放炸弹
7. 在窗口B同样操作
8. 观察两个窗口的实时同步效果！

## 方式三：与Bot对战（单人模式）

1. 启动服务器，打开浏览器
2. 输入名字，选择模式 **"人机挑战"**
3. 创建房间
4. 系统自动添加3个Bot
5. 点击 **准备** → **开始游戏**
6. 与Bot对战！

Bot行为：
- **简单**：随机走动，偶尔放炸弹
- **中等**：会躲避炸弹，会捡道具
- **困难**：会追击你，会封路
- **疯子**：贴脸放炸弹，非常激进

## 分享给外网好友（进阶）

如果你想让不在同一网络的朋友加入：

### 方案：使用 ngrok

1. 访问 [ngrok.com](https://ngrok.com) 注册免费账号
2. 下载 ngrok 客户端
3. 在终端中运行：
```bash
ngrok http 7100
```
4. 复制类似 `https://abcd-123-456-789.ngrok-free.app` 的链接
5. 把这个链接发给朋友，朋友打开即可直接游戏

> ⚠️ 免费版 ngrok URL 每次重启会变化，需要重新分享。

## 故障排查

| 问题 | 解决方案 |
|------|---------|
| `npm install` 失败 | 检查 Node.js 是否安装：`node -v` |
| 端口7100被占用 | 修改 `server.js` 第9行端口，或关闭占用程序 |
| 浏览器打不开 | 检查防火墙是否放行7100端口 |
| 两个窗口无法同步 | 确保两个窗口都连接了同一个服务器 |
| 游戏画面空白 | 检查浏览器控制台是否有报错（F12 → Console） |
| 炸弹不爆炸 | 检查后端服务器是否还在运行 |
